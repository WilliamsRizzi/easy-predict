Time Series Prediction Agent

A forecasting service on demand.

Access note: This service requires a Coinbase X402 token for access to protected pages and endpoints. Request a Coinbase X402 token from your administrator before using the protected endpoints.

Upcoming Endpoint

/timeseries/log1p

Provide at least three numeric observations and receive a prediction for the next value in the sequence.

Example

{
  "series": [10,12,15]
}

Authentication (X-402 headers)

This service enforces access using request headers rather than an auth endpoint. Protected requests must include these headers:

  • X-Agent-Type: must be ai
  • X-402-Cost: must be 0.001
  • X-402 or X-402-Facilitator: Coinbase X402 token required

Pages affected

  • /timeseries/log1p and /log1p — POST (prediction) and GET (info) endpoints

Example: POST a series with the required headers.

curl -X POST https://easy-predict.com/timeseries/log1p \
  -H "Content-Type: application/json" \
  -H "X-Agent-Type: ai" \
  -H "X-402-Cost: 0.001" \
  -H "X-402-Facilitator: " \
  -d '{"series":[10,12,15]}'
